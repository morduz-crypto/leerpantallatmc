# Tour rápido — Cómo ubicarse en pantallas (NetSuite)

Template estático (GitHub Pages) para capacitar sin aburrir.

## Reemplazar screenshots
En `/assets` reemplaza manteniendo los nombres:
- `saved-search.png`
- `report.png`
- `record.png`
- `errors.png`

## Ajustar pines
Edita `content.js`:
- `x` y `y` son porcentajes (0–100) sobre la imagen.
- `title` y `body` son el texto de cada pin.

## Publicar en GitHub Pages
- Repo → Settings → Pages
- Source: Deploy from a branch
- Branch: main
- Folder: / (root)
