# Tour rápido — Cómo leer pantallas (NetSuite)

Este mini-sitio es un template estático para GitHub Pages.
Objetivo: explicar de forma general cómo ubicarse en 4 pantallas universales:
1) Lista / Saved Search
2) Informe / Report
3) Registro (transacción / entidad / custom record)
4) Errores + warnings + permisos

## Cómo personalizar
1. Cambia las imágenes en `/assets` manteniendo el mismo nombre:
   - saved-search.png
   - report.png
   - record.png
   - errors.png

2. Edita textos y posiciones de pines en `content.js`.
   - `x` y `y` son porcentajes (0–100) sobre la imagen.

## Publicar en GitHub Pages
- Sube la carpeta al repo.
- Settings → Pages → Deploy from branch → `/root` → Save.
